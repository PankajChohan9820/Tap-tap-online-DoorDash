# Tap-tap-online-DoorDash

Need to have Python install
- Install jwt, requests

If you already have data.json file delete it first.


To run the code
To create the order python3 app.py order_no
Here order_no means 1st order or 2nd or 3rd order
e.g python3 app.py 2 (to create order for 2nd)

To get the status of order

python3 order_status order_no

e.g To get status of order no 3 run
python3 order_status 3

To cancel the last order just run

python3 cancel_order.py
